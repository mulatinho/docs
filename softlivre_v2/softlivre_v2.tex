	\framesubtitle{Qual a definição para \large{\textbf{software}}!?}
	
	\emph{"Qualquer programa ou grupo de programas que instrui o hardware sobre a 
	maneira como ele deve executar uma tarefa, inclusive sistemas operacionais,
	processadores de texto e programas de aplicação."} Dicionário Michaelis
	
	\vskip 5mm Ou ainda... \vskip 5mm
	
	\emph{"Uma série de instruções definidas por uma linguagem de programação que
	com uma lógica escrita por um ou mais desenvolvedor(es) executa operações em
	sistematicamente afim de se atingir um objetivo específico."}
\end{frame}

\begin{frame}
	\frametitle{Conceitos Básicos}
	\framesubtitle{Principais requisitos}
	
	O que é \textbf{necessário para construir} um software!?
	
	\begin{enumerate}
		\item Um hardware (computador, sistema embarcado, etc.)
		\item Um compilador (transformará o código em binário)
		\item Um desenvolvedor (responsável pelo código do programa)
	\end{enumerate}
	
	Geralmente existe um sistema operacional entre os itens 1 e 2, embora isso
	não seja necessariamente a regra.

\begin{frame}
	\frametitle{Conceitos Básicos}
	\framesubtitle{Software Livre}
	
	Todo o software que cumpre os quatro requesitos citados abaixo é conhecido
	como software livre. Os requesitos, também conhecidos nas comunidades como 
	liberdades, são garantias aos usuários. Eis as quatro liberdades:
	
	\begin{description}
		\item[nº 0:] Obter e executar o programa com qualquer propósito,
		seja ele acadêmico, comercial ou pessoal.
		\item[nº 1:] Estudar como o programa funciona e adaptá-lo
		as suas necessidades caso seja necessário. [*]
	\end{description}
	
	[*] O código-fonte é obrigatório para atender esta liberdade.
\end{frame}

\begin{frame}
	\frametitle{Conceitos Básicos}
	\framesubtitle{Software Livre, parte 2.}
	
	\begin{description}
		\item[nº 2:] Redistribuição de cópias do software obtidas, ou seja, 
		uma vez que você adquiriu um software você pode compartilhá-lo.
		\item[nº 3:] Modificação e redistribuição das cópias feitas	por você
		para os seus amigos, seus clientes, "sua comunidade". \textit{Bastando apenas
		preservar o nome dos autores originais}.
	\end{description}
	
	\begin{block}{}
		No softwares livre não existem donos, mas sim autores.
	\end{block}
	
	\vskip 5mm
	
	O autor é protegido por uma licença chamada \textbf{Copyleft}.
\end{frame}
